Edit DNS Record

To create a new DNS record, you must enter the following information:

  • The name of the record, including the full domain name (e.g., example.is404.net)
  • The type of record (e.g., A, CNAME, MX, TXT)
  • The content of the record (e.g., the IP address for an A record, or the target for a CNAME record)
  • The TTL (time to live) for the record (e.g., 1, 120, 3600). You can leave this at the default of 1.
